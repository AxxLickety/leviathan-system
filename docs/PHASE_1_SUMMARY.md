# Phase 1 — Summary (Regime dependence)

## Goal
Test whether an affordability proxy (DTI-based) behaves like a return factor or a risk filter, and whether behavior changes across regimes.

## Main finding (1 sentence)
Affordability behaves **regime-conditionally**: strong evidence of being a **risk/overheating filter** rather than a robust return-seeking factor.

## Interpretation
- In “down” environments the relationship is not a clean alpha signal.
- In “up/expansion” environments it acts more like a guardrail against left-tail outcomes / overheating.
